Android Card/Dice Game App

This is a game I made in HTML5, it works on both mobile and desktop web browsers and was created when mobile web browsers first started supporting hardware acceleration on the HTML5 canvas. The game uses hardware acceleration on any device capable of it, but falls back on software rendering to function on older devices. The gameplay is pretty simple but has an addictive element to it once you get going.

Before this project I already had experience with javascript, but this was my first time using the language for a larger and performance-critical application. I used performance profiling tools and some simple but effective tricks to get the game to run at a decent framerate even on older hardware without hardware-acceleration from a gpu and find + get rid of any memory leaks in my javascript. I learned how to architect a clean and understandable code structure with javascripts tricky prototypal-OOP, and how to apply the performance-conscious coding skills I learned from my earlier Java and C++ projects in javascript.
